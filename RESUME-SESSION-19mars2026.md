## 5. Structure juridique cible (roadmap)

### Phase 1 — Micro-entreprise (M0-M6, mars-septembre 2026)
- Transfert siège → 971
- Activer exonérations DOM
- Déposer dossiers subventions
- Premiers clients KCI + Topo3D

### Phase 2 — SASU + Holding (M6-M18)
- Quand CA > 77 700€ → créer SASU KCI
- Créer Holding SAS en Guadeloupe (ZFANG = abattement IS 80%)
- Filiales : KCI SAS + Topo3D Antilles SAS + ImmoServices971 SAS

### Phase 3 — SCI + Multi-DOM (M18+)
- SCI pour acquisitions immobilières (financée par les dividendes des filiales via la holding)
- Extension Martinique, Guyane, Réunion

### Flux financiers intra-groupe
Les filiales versent leurs bénéfices en dividendes à la holding (régime mère-filiale : exonération 95%). La holding réinvestit dans la SCI pour financer l'acquisition de biens immobiliers (apport + prêt bancaire). Les loyers remboursent le prêt.

## 9. Points clés à retenir

- **Tout sous un seul SIREN** : KCI, Topo3D et ImmoServices971 sont des noms commerciaux, pas des entreprises séparées
- **Exonération DOM 24 mois** : quasi zéro charges sociales → chaque euro de CA = euro dans la poche
- **Abattement IR 30% DOM** : ne pas opter pour le versement libératoire
- **Charges fixes : ~30-50€/mois** pour un potentiel de 50K€+/an de CA
- **Topo3D est le game changer** : meilleure marge (98%), 0 concurrent SaaS, marché captif (485 prospects)
- **Le réseau en Guadeloupe est le moteur de croissance** : famille = ambassadeurs, pros = premiers clients
- **Le volant d'inertie** : chaque client d'une activité est un prospect pour les deux autres
- **Dépenses perso remboursables** : les factures payées depuis le compte perso sont éligibles aux subventions
- **Holding = Phase 2** : permet de transférer les bénéfices vers une SCI pour acquérir de l'immobilier, quasi-exonéré (régime mère-filiale + ZFANG)
